Photography proofing plugin. Create secure client galleries with watermark, selection workflow and automated file renaming.

== Description ==

PhotoProof is a WordPress plugin for photographers. It allows you to create secure proof galleries for your clients, with watermark protection, client selection workflow, and automated file renaming.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/photoproof` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to Galleries > Settings to configure the plugin.
